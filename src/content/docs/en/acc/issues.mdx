This guide walks through how to find, review, and resolve issues that have been assigned to you in ACC.

## Find Your Assigned Issues

<Steps>
1. In ACC, click on **Issues** in the left sidebar to open the issues list.
   ![The ACC Issues page showing a list of open issues with the Issues nav item highlighted](/images/acc/acc_issues_step_1.png)

2. Review the list to find issues assigned to you. Each row shows the title, ID, status, type, and assignee.
   ![Issues list showing three issues with title, status, type, and assignee columns](/images/acc/acc_issues_step_2.png)
</Steps>

## Review the Issue in the Model

<Steps>
1. Click on an issue to open it. The viewer shows where the issue is marked on the model, with the details panel on the right.
   ![Model viewer with the Issues panel open and the issue annotation visible on the drawing](/images/acc/acc_issues_step_3.png)

2. The highlighted area on the canvas shows the exact location of the issue.
   ![Model viewer with the full drawing canvas highlighted to show the issue area](/images/acc/acc_issues_step_4.png)

3. Review the issue details — title, status, assignee, due date, and start date — in the right-hand panel.
   ![Issue details panel showing status, type, assignee, due date, and start date fields](/images/acc/acc_issues_step_5.png)
</Steps>

## Update the Issue Status

<Steps>
1. Open the issues list for the current model. You can see all related issues and their statuses.
   ![Issues panel listing multiple issues, one showing Pending status](/images/acc/acc_issues_step_6.png)

2. Click the **Status** field on the issue you are working on to edit it.
   ![Issue details panel with the Status field showing Open and the edit icon highlighted](/images/acc/acc_issues_step_7.png)

3. Select **In review** from the dropdown to indicate you are actively working on the issue.
   ![Status dropdown open with In review option highlighted](/images/acc/acc_issues_step_8.png)

4. The issue list now reflects the updated **In review** status.
   ![Issues panel showing the issue status updated to In review](/images/acc/acc_issues_step_9.png)
</Steps>

## Fix the Issue in Revit

<Steps>
1. Open **Autodesk Revit** and navigate to the floor plan or view where the issue was reported.
   ![Autodesk Revit open showing the floor plan for the relevant level](/images/acc/acc_issues_step_10.png)

2. In the Revit home screen, click **Autodesk Projects** in the left panel to browse cloud projects.
   ![Revit home screen with Autodesk Projects highlighted in the left sidebar](/images/acc/acc_issues_step_11.png)

3. Navigate to the project's **Links** folder and select the model that needs to be updated.
   ![Revit home showing the Links folder for the project with the relevant model highlighted](/images/acc/acc_issues_step_12.png)

4. Right-click the model and select **Publish latest without links** to push your changes back to ACC.
   ![Right-click context menu on the model with Publish latest without links highlighted](/images/acc/acc_issues_step_13.png)
</Steps>
