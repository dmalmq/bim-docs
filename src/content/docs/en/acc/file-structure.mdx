Our ACC projects follow a consistent folder structure across all projects.

## Top-Level Folders

| Folder           | Purpose                                                    |
| ---------------- | ---------------------------------------------------------- |
| `01_WIP`         | Work-in-progress models                                    |
| `02_Shared`      | Models shared for coordination                             |
| `03_Published`   | Deliverables published for construction                    |
| `04_Reference`   | Survey data, client documents, and site photos             |
| `05_Templates`   | Revit templates and family libraries                       |

<Aside type="caution">
  Do not save directly to **03_Published**. All deliverables must first go
  through the review process in `02_Shared`.
</Aside>

## Creating a New Project

<Steps>
1. Create the project in the ACC admin panel and assign members.
2. Copy the folder template from the *BIM Standards* project using the
   "Copy Folder Structure" feature.
3. Upload the latest Revit template from `05_Templates` into the new
   project's `01_WIP` folder.
4. Configure permissions — restrict `03_Published` to project leaders only.
</Steps>

<Aside type="tip">
  For projects using shared coordinates, complete the
  [Coordinate Setup Guide](/en/revit/new-project-setup/) before starting any
  modelling.
</Aside>
