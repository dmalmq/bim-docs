ACCプロジェクトでは統一されたフォルダ構成を採用しています。

## トップレベルフォルダ

| フォルダ           | 用途                                         |
| ------------------- | -------------------------------------------- |
| `01_WIP`           | 作業中のモデル                                |
| `02_Shared`        | コーディネーション用に共有されたモデル          |
| `03_Published`     | 施工用に発行された成果物                       |
| `04_Reference`     | 測量データ、クライアント資料、現場写真          |
| `05_Templates`     | Revitテンプレートとファミリライブラリ           |

<Aside type="caution">
  **03_Published** に直接保存しないでください。すべての成果物は、まず
  `02_Shared` のレビュープロセスを経る必要があります。
</Aside>

## 新規プロジェクトの作成

<Steps>
1. ACC管理画面でプロジェクトを作成し、メンバーを割り当てます。
2. *BIM Standards* プロジェクトから「フォルダ構成をコピー」機能でフォルダ
   テンプレートをコピーします。
3. `05_Templates` から最新のRevitテンプレートを新規プロジェクトの
   `01_WIP` フォルダにアップロードします。
4. 権限を設定します — `03_Published` はプロジェクトリーダーのみに制限します。
</Steps>

<Aside type="tip">
  共有座標を使用するプロジェクトの場合は、モデリングを開始する前に
  [座標設定ガイド](/ja/revit/new-project-setup/) を完了してください。
</Aside>
